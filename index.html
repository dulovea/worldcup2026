⚙️ Configuration Firebase

Pour activer la synchronisation temps réel entre tous vos joueurs, connectez un projet Firebase gratuit.
  1. Allez sur console.firebase.google.com
  2. Créez un projet → activez Realtime Database
  3. ⚙️ Paramètres → Vos applications → Web → copiez la config
  4. Règles DB → mettez .read:true, .write:true

Config sauvegardée localement dans votre navigateur.
